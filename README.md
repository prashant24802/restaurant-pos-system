# Restaurant POS System

A full-stack Restaurant Point of Sale (POS) application built using:

## Tech Stack

### Frontend
- Angular
- Angular Material
- Tailwind CSS

### Backend
- Spring Boot
- Spring Security
- Spring Data JPA
- MySQL
- JWT Authentication (In Progress)

## Features

- User Authentication
- Menu Management
- Table Booking
- Kitchen Dashboard
- Billing & GST Invoice
- Daily Sales Report

🚧 Currently under development.
